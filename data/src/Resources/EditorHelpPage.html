1. File: the "Convert" in "File" menu will open PGDSpider for users to conver the file type.

2. Split: there are several split methods provided in "Split" menu if the file is not in proper format, the file could be split into columns by comma, space, tab, semicolon, line (If the file's type is xlsx or xls, the split menu is hidden).

3. General functions:

  • Replace missing data: replace the missing data with the given data.
  • Replace spaces in column: replaces all spaces with underscores in the given column.
  • Move column: move the given column to the end of current file.
  • Delete every second headers: delete every second cell in the given row.
  • Add text to column: add the given text to the end of every cell in the given column.

4.Remove columns/rows: the drop down boxes allow users to select one column/row each time. The two input boxes below allow users to select all columns/rows in the given range. The button "Seect rows by text sequences" allow users to select rows by providing a column index and a text sequent. All the selected columns/rows will show in the table ("Columns/Rows to remove"). Click "Delete" to remove the selected column/row from the table("Columns/Rows to remove").

* After doing 3 and 4, users need to click "Save" to see the changes happened on the right side data table.

5. Execute buttons

  • Save: save the file and all the changes will be made in the file.
  • Save as: save the file with a new name and the original file will be keeped, but the file's type can't be changed.
  • Clear: clear all changes(at the left side of editor window) that haven't been saved yet.
  • Fast convert: convert the current file from STRUCTURE to GENEPOP without opening the PGDSpider graphic user interface. This function will ask users to provide the size of the repeated motif, missing data and the number of markers(loci), if the user didn't input anything, the missing value will be 0 by default and the program will calculate the number of markers(loci) using the number of columns. The output file location would be the same as the input file location by default, users could also set the output file location by clicking "Destination Folder".(Go to the buttom of the page to see the spid file for fast convert.)
  • Close: close the editor window, please make sure all changes have been saved before closing.

* values in every cell can be selected and edited by double clicking the cell, but the value after edited won't be saved. This function is only used for copy.

* If the log file location has been setted, all the changes happened in the file will be recored in the "logEdit.txt".

* To use PGDSpider, you must have Java Runtime Environment installed.

# STRUCTURE Parser questions
PARSER_FORMAT=STRUCTURE
# What is the ploidy of the data?
STRUCTURE_PARSER_PLOIDY_QUESTION=DIPLOID_ONE_ROW

# Select the type of the data:
STRUCTURE_PARSER_DATA_TYPE_QUESTION=MICROSAT

# Enter the number of markers (loci) listed in the input file:
STRUCTURE_PARSER_NUMBER_LOCI_QUESTION=enter by user

# Are marker (locus) names included?
STRUCTURE_PARSER_LOCI_NAMES_QUESTION=false

# How are Microsat alleles coded?
STRUCTURE_PARSER_MICROSAT_CODING_QUESTION=REPEATS

# Enter the size of the repeated motif (same for all loci: one number; different: comma separated list (e.g.: 2,2,3,2):
STRUCTURE_PARSER_REPEAT_SIZE_QUESTION=enter by user

# What is the missing value code (-9, -999, ...):
STRUCTURE_PARSER_MISSING_CODE_QUESTION=enter by user

# Is the "PopData" column (population identifier) present in the input file?
STRUCTURE_PARSER_POP_DATA_PRESENT_QUESTION=true

# Is the "Phase Information" row present?
STRUCTURE_PARSER_PHASE_ROW_QUESTION=false

# Are individual names (labels) included in the input file?
STRUCTURE_PARSER_IND_NAMES_QUESTION=false

# Are the "Recessive Alleles" row and/or the "Inter-Marker Distance" row present in the input file?
STRUCTURE_PARSER_ADDITIONAL_ROW_QUESTION=NONE

# GENEPOP Writer questions
WRITER_FORMAT=GENEPOP

# Specify which data type should be included in the GENEPOP file (GENEPOP can only analyze one data type per file):
GENEPOP_WRITER_DATA_TYPE_QUESTION=MICROSAT

# Specify the locus/locus combination you want to write to the GENEPOP file:
GENEPOP_WRITER_LOCUS_COMBINATION_QUESTION=
