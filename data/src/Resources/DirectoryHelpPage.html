1. File: set the log file location.

2. Editor: open a new window and edit the file's foramt.

3. Open: open a file with it's default application.

4. Locate: open the folder that contain the file.

5. Decompress: decompress zip, tar file, if the folder has already exist, the new folder will cover the old one.

6. Delete: delete the file permanently.

7. Move to Trash: move the file to trash

8. Meta Data tab show all meta data files in the current folder.

9. README tab show all README files in this folder.

10. The "Icon" and "File" labels in the left side file table allow users to sort files by type or by name.


* If the log file location has been setted, all the delete action will be record in the "logDelete.txt".
