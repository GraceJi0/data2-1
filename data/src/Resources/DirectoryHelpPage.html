1. File: set the log file location.

2. Open: open a file with it's default application.

3. Locate: open the folder that contain the file.

4. Editor: open a new window and edit the file's foramt.

5. Decompress: decompress zip, tar file.

6. Delete: delete the file permanently.

7. Meta Data tab show all meta data files in the current folder.

8. README tab show all README files in this folder.

9. The "Icon" and "File" labels in the left side file table allow users to sort files by type or by name.
